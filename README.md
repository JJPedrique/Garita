# Garita
Proyecto de ingenieria del Software I y II, Sistema de Administracion de Control de Acceso de una Asociacion de Vecinos

### Idea Principal
El sistema debe permtir el acceso a la vivienda de manera automatica si estas solvente con el pago, de ser el caso contrario, deberar abrir el porton de manera manual. En la tabla <b>registros</b> se alamcenan todas los pago que una vivienda ha realizado, la cantidad de pagos realizaodos debe coincidir con la cantidad de <b>pagos</b> que se han emitido (o guardados en la tabla de pagos), de lo contrario esta "moroso"
